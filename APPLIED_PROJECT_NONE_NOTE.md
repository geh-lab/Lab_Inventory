# Project Relation None Patch

적용일: 2026-05-29

## 반영 내용

- 과제 연관성 옵션에 `없음`을 추가했습니다.
- 물품 목록 상단 과제 연관성 필터에 `없음` 칩이 표시됩니다.
- 물품 상세/수정 모달의 과제 연관성 선택 영역에도 `없음` 버튼이 표시됩니다.
- 신규 물품 등록 시 과제 연관성을 선택하지 않으면 `없음`으로 저장됩니다.
- 기존 데이터 중 과제 연관성 값이 비어 있거나 `없음`, `해당 없음`, `미지정`, `none`, `n/a` 등으로 저장된 경우 모두 `없음`으로 정규화합니다.
- `노지` 필터는 기존처럼 `노지 + 모두`, `온실` 필터는 `온실 + 모두`를 포함합니다.
- `모두` 필터는 `모두`로 저장된 물품만 표시합니다.
- `없음` 필터는 과제 연관성이 없는 물품만 표시합니다.
- 과제 연관성 옵션이 4개가 되면서 모달 버튼 배열이 깨지지 않도록 `.segmented` 그리드를 보정했습니다.
