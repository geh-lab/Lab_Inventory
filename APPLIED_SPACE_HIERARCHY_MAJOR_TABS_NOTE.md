# Space Hierarchy Major Tabs 적용 메모

## 적용 내용

1. 공간 맵 상단 탭을 개별 공간 목록이 아니라 `대분류` 목록으로 변경했습니다.
   - 예: `농업생명과학대학`, `KT&G`, `기타`처럼 대분류만 표시됩니다.
   - 대분류 탭을 누르면 해당 대분류에 속한 공간 카드만 공간 맵에 표시됩니다.
   - 공간 카드 자체를 누르면 오른쪽 상세 패널에는 해당 공간의 물품만 표시됩니다.

2. 공간 추가/수정 모달에서 기존 계층을 재사용할 수 있게 했습니다.
   - `대분류`, `중분류`, `소분류` 입력칸에 기존 값이 자동완성 목록으로 뜹니다.
   - 입력칸 아래 `기존 계층 목록` 영역에서 기존 값을 클릭하면 바로 입력됩니다.
   - 이미 생성한 대분류를 새 공간에 다시 입력할 때 중복으로 새 체계를 만들지 않고 기존 값을 그대로 사용할 수 있습니다.

3. 저장 로직을 정리했습니다.
   - 공간 저장 시 `major`, `middle`, `minor`가 정리된 값으로 저장됩니다.
   - `group` 값은 기존 데이터 호환을 위해 `major`와 같은 값으로 유지됩니다.
   - 공간을 저장하면 공간 맵의 선택 대분류와 선택 공간이 바로 저장한 공간으로 이동합니다.

4. 캐시 무효화 버전을 변경했습니다.
   - `styles.css?v=hierarchy-major-tabs-20260529`
   - `app.js?v=hierarchy-major-tabs-20260529`

## 확인 방법

1. 관리자 계정으로 로그인합니다.
2. `공간 추가` 또는 `공간 수정`을 엽니다.
3. 대분류에 기존 값이 자동완성/칩으로 뜨는지 확인합니다.
4. 저장 후 `공간 맵` 탭에서 상단 탭이 개별 공간이 아니라 대분류 단위로 보이는지 확인합니다.
