# styles.css 패치 적용 안내 (2026-05-28)

## 무엇이 바뀌었나
라이트 모드의 가독성·삭제 버튼·공간 카드 색상 문제를 보정하고,
다크/라이트 공통으로 포커스 가시성·삭제 버튼 호버·모바일 사이드바 토글 위치를 손봤습니다.

### 주요 수정 사항
1. **삭제(Danger) 버튼**
   - 라이트 모드: 흰 글자 + 빨강 그라데이션으로 변경해 글자/아이콘이 확실히 보입니다.
   - 다크 모드: 호버 시 글자 흰색으로 또렷하게.
2. **공간 맵 카드 (라이트 모드)**
   - 흰 카드 베이스 + 좌측 4px 컬러 스트라이프로 온실/실험실/공용을 한눈에 구분.
   - 활성 상태는 outline 으로 표시 (배경색 변형 X → 글자 가독성 유지).
3. **미니 맵·아이콘·칩·태그**
   - 아이콘 배경을 컬러 톤별 파스텔로 통일, 글자는 진한 슬레이트로.
   - 칩 active 색을 흰배경 + 진한 텍스트로 변경해 튀지 않게.
4. **카드/패널/모달/입력 폼**
   - 라이트 모드에서 일관된 #ffffff 카드 + soft 그림자.
   - 입력란 placeholder, 포커스 링, qty 버튼 톤 정리.
5. **포커스/접근성**
   - 키보드 포커스 시 outline 강제로 표시 (`:focus-visible`).
6. **모바일 (≤620px)**
   - 사이드바 숨김 시에도 좌상단에 햄버거 버튼이 떠 있도록 위치 고정.

## 적용 방법
**옵션 A — 가장 빠른 방법:** `index.html` 의 `<link rel="stylesheet" href="styles.css?...">` 바로 아래에 한 줄 추가.

```html
<link rel="stylesheet" href="styles.css?v=2026-05-27" />
<link rel="stylesheet" href="styles.patch.css?v=2026-05-28" />
```

같은 폴더에 `styles.patch.css` 를 복사해 넣으면 끝입니다.

**옵션 B — 한 파일로 합치기:** `styles.patch.css` 의 내용 전체를
기존 `styles.css` 의 **맨 아래**에 붙여넣어도 됩니다.
(패치는 동일 셀렉터를 더 강한 우선순위로 덮어쓰도록 작성되어 있어 마지막에 위치하면 됩니다.)
